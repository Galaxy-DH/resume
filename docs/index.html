应用技术：线程池、epoll、Socket、TCP、MySQL
项目介绍：基于 Linux 平台开发，采用 Reactor 多线程 + epoll 监听架构，保证企业员工借助账号管理机制通过该服务器实现文件传输。 企业员工可将个人工作相关文件保存至企业服务器，便于员工连接服务器下载、上传相关文件。
责任模块：主要负责实现用户注册登录、日志记录、文件传输
主要工作：
  • 使用 MySQL 存储用户登录信息，实现用户登录验证
  • 记录日志信息：客户端请求、操作记录及时间，将之存放在数据库中
  • 拓展使用 mmap 优化大文件传输，实现上传、下载文件，支持下载断点续传
技术亮点：
  • 采用虚拟文件表设计，实现用户拥有各自独立的目录结构
  • 生成并检查文件 md5，实现不同用户相同文件秒传

某企业内部信息检索工具——【项目组核心成员】(2020.11 - 2021.04)
应用技术：Reactor 多线程、Json、Redis、Simhash、TF-IDF 算法
项目介绍：基于 Linux 平台开发，实现企业内部产品、文件及相关资料信息的检索功能。项目分为两个模块，定义客户端与服务端的消息格式，实现对不同模块的调用。模块一，根据用户输入的信息进行联想推荐并返回结果。 模块二，进行检索后返回相关文件的基本信息（标题，摘要，链接），实现中英文检索功能。
责任模块：负责模块二的网页去重、建立倒排索引以及查询返回实现
主要工作：
  • 采用 Reactor 多线程搭建服务器框架
  • 使用cppjieba分词库、Simhash 算法，对所提供的网页库进行网页去重
  • 运用 TF-IDF 算法，对去重后的网页库建立倒排索引库
  • 运用余弦相似度算法，通过倒排索引库对匹配查询词的网页进行排序，并使用 jsoncpp 封装查询结果后返回给客户端
技术亮点：
  • Simhash 算法（网页去重），TF-IDF算法(倒排索引)
  • UTF-8中英文编码处理，单例模式，最小编辑距离算法
个人收获：对于 Reactor 网络模式实现高并发有了更清晰的理解，同时对于搜索引擎技术有了新的认识，开拓了视野。
